# Usage

### Make virtual environment

```console
uv venv venv -m 3.10
```

or 

```console
python -m venv venv
```

then source

```console
source venv/bin/activate
```

then

```console
uv pip install -r requirements.txt
```

or 
```console
pip install -r requirements.txt
```



then 


```console
python main.py
```

all game config vars in config file for best practice
